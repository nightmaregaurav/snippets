Step to create Oracle Database 18c Express edition docker image
     git clone https://github.com/oracle/docker-images.git
     cd docker-images
     cd OracleDatabase/SingleInstance/dockerfiles
     sudo ./buildContainerImage.sh -h
     (Use flags as asked) example: sudo ./buildContainerImage.sh -v 18.4.0 -x
     Wait for really long tome

Create local mount points:
     mkdir -p ~/DockerShare/odb18xe/oradata/
     mkdir -p ~/DockerShare/odb18xe/scripts/setup/
     mkdir -p ~/DockerShare/odb18xe/scripts/startup/

Change permission so amyone could write there:
     sudo chmod -R 777 ~/DockerShare/odb18xe/
     

To run oracledb docker:
     docker run \
     -itd \
     --hostname odb18xe \
     --name odb18xe \
     -p 1521:1521 \
     -p 5500:5500 \
     -v ~/DockerShare/odb18xe/oradata:/opt/oracle/oradata \
     -v ~/DockerShare/odb18xe/scripts/setup:/opt/oracle/scripts/setup \
     -v ~/DockerShare/odb18xe/scripts/startup:/opt/oracle/scripts/startup \
     oracle/database:18.4.0-xe

To resume container: docker start odb18xe

ORACLE PASSWORD FOR SYS, SYSTEM and PDBADMIN accounts and connection strings will be given in top of: docker logs odb18xe

To get terminal: docker exec -it --user=oracle odb18xe bash

Set up the required Linux environment variables for Oracle Database: oraenv
input: XE


To enter database sql shell from inside docker terminal: sqlplus / as sysdba
To enter pluggable database sql shell from inside docker terminal: sqlplus sys@XEPDB1 as sysdba
